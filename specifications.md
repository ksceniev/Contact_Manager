Sistema de Contatos:

* Todos os dados adicionados e editado devem ser gravados em um arquivo
* Os dados para cada contato devem incluir:
	* Nome
	* Telefone
	* email
	* endereço
	
*As opções da interface são:
	* Adicionar novo contato
	* Listar todos os contatos
	* Procurar contato
	* Editar contato
	* Deletar contato
